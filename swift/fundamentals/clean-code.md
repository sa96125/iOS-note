### Constant File

String 데이터를 사용할 때 주의할 점은 오타로 인해 발생하는 오를 Xcode는 대처해주지 않는다는 점이다. 따라서 항상 문자열을 사용해야하는 경우, constatant 파일에 기입하여 후에 발생될 수 있는 에러를 방지한다.
