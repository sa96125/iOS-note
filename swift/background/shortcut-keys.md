# Shortcut keys

|                      |                            |
| -------------------- | -------------------------- |
| new File             | `command + n`              |
| build                | `command + r`              |
| rename variable      | `command + click → Rename` |
| code block folding   | `command + alt + <-`       |
| 스토리 보드 엘리먼트 복사하기     | `alt + Drag`               |
| 스토리보드 @IBOutlet 연결하기 | `control + Drag`           |
| reindent             | `control + i`              |
